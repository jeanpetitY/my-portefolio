# My Portfolio

Personal research and engineering portfolio built with [Rasengan.js](https://rasengan.dev/).

It presents my experience, projects and publications across artificial intelligence, human-computer interaction, knowledge graphs and scholarly infrastructure.

## Development

```bash
npm install
npm run dev
```

Production build:

```bash
npm run build
```

## Deploy to Vercel

The official Rasengan Vercel adapter is already configured. Import the Git repository into Vercel or run:

```bash
npx vercel --prod
```

## Stack

Rasengan.js · React · TypeScript · CSS · Vercel
